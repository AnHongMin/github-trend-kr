vim
==================================================
- 페이지 링크: https://github.com/vim/vim

이 프로젝트는 새로운 뭔가가 아니라 기존에 code.google.com 에 있는 vim의 소스를 가져온 거라고 합니다.
이제는 vim이 vi를 거의 대체 해버려서 둘의 차이점 같은건 잘 모르고 있었는데
프로젝트 페이지에 간단하게 나와 있습니다.

* 멀티 레벨 undo
* 문법 강조
* 커맨드 라인 히스토리
* 온라인 도움말
* 스펠 체크
* 파일 이름 자동완성
* 블럭 오퍼레이션
* GUI
