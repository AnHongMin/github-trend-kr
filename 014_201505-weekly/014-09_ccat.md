# ccat
- 페이지링크: [ccat](https://github.com/jingweno/ccat)

ccat는 cat를 색상화한 것입니다. cat와 비슷하게 동작하지만, syntax highlighting을 하여 content를 보여줍니다. 

JavaScript, Java, Ruby, Python, Go, C 등의 언어를 지원해준다고 하네요..
